Data and Knowledge Research Group

The Data and Knowledge Research Group at UNSW is world leading in database, data management, and (big) data analytics. Our research accelerates data-intensive applications, and supports insightful knowledge extraction from large-scale, complex and dynamic real-life datasets. Our investigation covers graph/network data, spatial-temporal data, relational data, multi-media data, text data, and more.

Research Interests

Graph Data Analytics and Management

Include data mining, data streams, distributed database systems, spatial database systems, and web databases

Multi-media Data Processing

Include data mining, data streams, distributed database systems, spatial database systems, and web databases

AI4DB / DB4AI

Machine learning algorithms for graph structured data and database systems, including developing graph neural network architectures and applying machine learning techniques to improve the algorithms in database domain.

Spatial-Temporal Data Processing

Spatial keyword search, spatial-social data processing

Text Mining

The process of transforming unstructured text into a structured format to identify meaningful patterns and new insights
